Aqui está a explicação de **como ler e alterar dados de um workbook do Excel** resumida em **passo a passo simples**:

---

# Passo a passo: Ler e alterar dados de um Excel com OpenPyXL

### 1. Separar arquivos para criar e ler

* Crie um arquivo `creating.py` para **criar** o workbook.
* Crie outro arquivo `reading.py` para **ler e alterar** o workbook.
* Isso evita sobrescrever dados acidentalmente.

---

### 2. Abrir um workbook existente

```python
from openpyxl import load_workbook

excel_file = "workbook.xlsx"
workbook = load_workbook(excel_file)
```

---

### 3. Selecionar a planilha desejada

* Para pegar a planilha ativa:

  ```python
  worksheet = workbook.active
  ```
* Para pegar uma planilha específica:

  ```python
  worksheet = workbook["MinhaPlanilha"]
  ```

---

### 4. Iterar pelas linhas e colunas

* Ler os dados linha por linha:

  ```python
  for row in worksheet.iter_rows(min_row=2):  # começa na linha 2 para ignorar cabeçalhos
      for cell in row:
          print(cell.value, end="\t")
      print()
  ```

* Você pode definir **linha/coluna inicial e final**:

  ```python
  for row in worksheet.iter_rows(min_row=2, max_row=10, min_col=1, max_col=3):
      ...
  ```

---

### 5. Alterar valores específicos

* Pelo endereço da célula:

  ```python
  worksheet["B3"].value = 14  # muda a idade da Maria
  ```
* Ou usando uma lógica para encontrar o valor:

  ```python
  for row in worksheet.iter_rows(min_row=2):
      if row[0].value == "Maria":  # coluna 0 = nomes
          row[1].value = 23       # coluna 1 = idade
  ```

---

### 6. Salvar alterações

```python
workbook.save(excel_file)
```

---

✅ Com isso, você consegue:

* Ler qualquer workbook.
* Navegar pelas planilhas.
* Iterar por linhas e colunas.
* Alterar dados específicos.
* Salvar o arquivo atualizado.
