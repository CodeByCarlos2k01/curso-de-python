# Passo a passo — Trabalhando com listas mutáveis no Python

1. **Listas são mutáveis**

   * Você pode alterar, adicionar ou remover elementos depois de criar a lista.
   * Exemplo:

     ```python
     lista = [10, 20, 30, 40]
     ```

2. **Acessando elementos**

   * Use índices para acessar valores.

     ```python
     print(lista[2])  # 30
     ```
   * Também é possível usar índices negativos (`-1` = último elemento).

3. **Alterando elementos**

   * Substitua o valor em um índice específico:

     ```python
     lista[2] = 300
     print(lista)  # [10, 20, 300, 40]
     ```

4. **Removendo elementos**

   * `del lista[indice]` apaga um elemento e reorganiza os índices.

     ```python
     del lista[2]
     print(lista)  # [10, 20, 40]
     ```
   * Atenção: remover elementos no meio de listas grandes exige muito processamento.

5. **Adicionando elementos no final**

   * Use `append()` para adicionar um item no final da lista:

     ```python
     lista.append(50)
     print(lista)  # [10, 20, 40, 50]
     ```

6. **Removendo o último elemento**

   * Use `pop()` para remover e retornar o último item:

     ```python
     ultimo = lista.pop()
     print(ultimo)  # 50
     print(lista)   # [10, 20, 40]
     ```

7. **Removendo um elemento específico**

   * `pop(indice)` remove o item do índice escolhido:

     ```python
     lista.pop(1)
     print(lista)  # [10, 40]
     ```

8. **Dicas de eficiência**

   * Evite adicionar ou remover elementos no início ou meio de listas grandes.
   * Para listas grandes, mexa preferencialmente no **final**.
   * Para operações frequentes no começo ou no meio, considere usar outros tipos de dados (ex.: `deque`).

9. **Resumo do CRUD em listas**

   * **C**reate: `append()` ou `insert()`
   * **R**ead: acessar por índice (`lista[2]`)
   * **U**pdate: alterar valor (`lista[2] = novo_valor`)
   * **D**elete: `del lista[indice]` ou `pop()`
